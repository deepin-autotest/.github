## Hi 来了~ 👋

Funny-Test（有趣的测试）是希望能聚拢一群“臭味相投”的人；

我们的目标是构建一些有用的自动化测试工具，不断探索自动化测试领域的前沿技术，解决自动化测试各领域的难点。

如果你想了解，可以查看我们的仓库，也可以查看我们的 wiki：https://funny-test.github.io/funny-docs
